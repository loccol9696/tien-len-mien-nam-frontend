# Thư mục Public

Thư mục này chứa các file tĩnh cho website.

## Thêm nhạc Tết

Để thêm nhạc Tết vào website, hãy đặt file nhạc vào thư mục này với tên `tet-music.mp3`.

### Các định dạng được hỗ trợ:
- MP3 (khuyến nghị)
- OGG
- WAV

### Cách thêm:
1. Tải file nhạc Tết về máy
2. Đổi tên file thành `tet-music.mp3`
3. Đặt file vào thư mục `public/` này
4. Khởi động lại server (nếu đang chạy)

### Gợi ý nhạc Tết:
- Nhạc Tết truyền thống Việt Nam
- Nhạc Xuân vui tươi
- Nhạc chúc Tết

File nhạc sẽ tự động được phát khi người dùng nhấn nút nhạc trên website.
